# Cypress: Automatizando testes E2E
Este é um repositório com a implementação prática do curso "Cypress: automatizando testes E2E" da Alura, ministrado pela instrutora Camila Pessôa.

## Sobre o curso
O curso aborda os principais conceitos e práticas para a criação de testes automatizados *end-to-end* com Cypress, uma ferramenta poderosa e fácil de usar. Aprenderemos a utilizar a interface gráfica do Cypress e também executá-los via terminal.

## Conteúdo do repositório
Este repositório conterá os arquivos de testes desenvolvidos ao longo do curso.

## Pré-requisitos
Antes de começar, é importante ter:
*   A <a href="https://cursos.alura.com.br/formacao-programacao" target="_blank">**formação Iniciante em Programação**</a> da Alura.
*   Noções sobre **desenvolvimento web**, como ***Status Code***, **métodos HTTP** e **API**.
*   Conhecimentos básicos em **HTML**, **CSS**, **JavaScript** e **DOM**.

## Como executar os testes (Em breve)
As instruções detalhadas para executar os testes serão adicionadas conforme avançamos no curso.
